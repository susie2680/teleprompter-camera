# 提示词相机

一个手机优先的网页版提词录像原型。

## 文件

- `index.html`：页面入口
- `style.css`：移动端界面样式
- `app.js`：摄像头、提词器滚动、录制下载逻辑

## 使用方式

手机浏览器调用摄像头通常要求 HTTPS。最稳妥的试用方式是把这个目录上传到任意 HTTPS 静态托管服务，例如 Vercel、Netlify、GitHub Pages 或自己的服务器。

电脑本机可以在目录里运行：

```bash
python3 -m http.server 8080
```

然后打开：

```text
http://localhost:8080
```

## 当前版本说明

- 默认调用前置摄像头和麦克风。
- 文案显示在预览画面上方，录制文件默认不包含文案。
- 支持调节滚动速度、字号、提示区域透明度和提词位置。
- 录制时会按当前手机方向导出：竖屏录制为竖屏视频，横屏录制为横屏视频。
- 支持镜像预览，但录制视频默认不包含提词文案。
- 浏览器支持不同，导出格式可能是 `MP4` 或 `WEBM`。
